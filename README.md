## Desarrollo

1. **Clonar repo**

  ```
  $ git clone https://github.com/CodeXtinction/meetupjs-app
  ```

2. **Instalar dependencias**

  ```
  $ yarn
  ```
3. **Correr app**

  ```
  $ react-native run-android
  ```
